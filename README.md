# 🧩 Blind 75 - Algoritmos & Estruturas de Dados

Este repositório contém minha jornada de estudos através do **Blind 75** (via [NeetCode.io](https://neetcode.io/)), focado em dominar os padrões de lógica e estruturas de dados mais exigidos em processos seletivos de tecnologia.

O objetivo principal aqui não é apenas encontrar o "Accepted" no LeetCode, mas sim entender a transição entre soluções de **Força Bruta (O(n^2))** e soluções **Otimizadas (O(n))**.

## 🚀 Minha Metodologia

Para cada desafio, sigo este fluxo de aprendizado:
1. **Abordagem Inicial:** Tento resolver o problema em até 30 minutos usando lógica pura (geralmente gerando soluções de força bruta).
2. **Análise de Complexidade:** Identifico os gargalos da solução inicial usando a notação **Big O**.
3. **Otimização:** Estudo e implemento a versão otimizada utilizando estruturas como `HashSets`, `HashMaps` ou técnicas de `Pointers`.
4. **Documentação:** Posto ambas as versões para registrar a evolução do raciocínio.
